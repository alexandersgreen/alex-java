Numerical Expressions

The Numerical Expression components are all expressions that have a numerical value when evaluated. The basic numerical expressions available are addition (+), subtraction (-), multiplication (* or .), division (/), and integers. Other numerical expressions can be constructed by using any of the given basic expressions, along with brackets. Within the editor, to construct a numerical expression the user simply types (in an available command line) the expression using the given binary operators, followed by a semi-colon (;), with spaces between each operator and int value. When entered the system creates the expression, and displays it accordingly. Edit operations can be performed on these expressions (and there sub expressions), as well changing their font and colour.
